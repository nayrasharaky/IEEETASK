# IEEETASK
# JavaScript Tasks Project

This project contains 3 simple JavaScript tasks that demonstrate basic programming concepts such as strings, arrays, loops, and problem solving.

---

## 📌 Task 1: Text Analysis
This task takes a sentence from the user and performs the following:
- Counts the number of words
- Counts letters without spaces
- Finds the most repeated word

### Example:
Input:
Hello world hello

Output:
Number of words: 3  
Letters without spaces: 15  
Most repeated word: hello  

---

## 📌 Task 2: Array Processing
This task works on an array of numbers:
- Removes duplicate values
- Sorts numbers in ascending order
- Calculates the sum of unique numbers

### Example:
Input:
[5, 2, 8, 2, 3, 5, 1]

Output:
Sorted numbers: [1, 2, 3, 5, 8]  
Sum: 19  

---

## 📌 Task 3: Flatten Array
This task converts a 2D array into a 1D array.

### Example:
Input:
[[1,2],[3,4],[5,6]]

Output:
Final Array: [1,2,3,4,5,6]
